# OpenCV Android Project

### Как запустить локально:

1. Откройте терминал в корне проекта.
2. Запустите скрипт настройки (скачивает SDK):
   `sh setup_opencv.sh` (macOS/Linux)
   *На Windows запустите аналогичные команды из скрипта вручную или используйте Git Bash.*
3. Откройте проект в Android Studio.
4. Нажмите **Sync Gradle**.
5. Запустите на устройстве.

### CI/CD:
В папке `.github/workflows` лежит скрипт, который автоматически скачивает OpenCV при сборке на GitHub.
